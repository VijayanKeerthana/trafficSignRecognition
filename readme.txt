This is road sign recognition project in machine learning.

The folder training code contains two parts:
1. traffic_sign_detection
	Contains the google collab code for training the detection model based on yolo. 
2. traffic_sign_classification
	Contains the code for training the classification model and the code to detect the bounding box values from yolo output and give the value to the input of classification model.
